How to play on the mc.shitfuck.pro Minecraft server:

  • The IP: mc.shitfuck.pro (no need to add port number).
  • Whitelist: Contact me on my socials or email kerbal255@outlook.com for whitelist.
  • Game version: Mineraft Java 1.20.2 (because some server-side performance mods aren't compatible with 1.20.4 yet).
  • No Chat Reports mod: The server has it, it's not required on your side, but it's highly recommanded.

    • Rules:

  • Freecam is allowed but only for working on your own buildings/farms, don't use it for PVP.
  • Placement mods are allowed but obviously to some degree, so bridging modes are allowed but stuff like Litematica Printer is obviously not ok.
  • Don't mess with other's buildings.

    • Recommanded mods:

    (Make sure to install the 1.20.2 version!)

    (Forge* means the mod does not come with Forge support but there is an unofficial Forge port).

  • No Char Reports (Fabric, Forge): Disable chat reports.
  • Mod Menu: Adds mod menu to Fabric.
  • Sodium (Fabric, Forge*): Massively enhancing performance.
  • Sodium Extra (Fabric, Forge*): Detailed render settings like fog distance.
  • Enhanced Block Entities (Fabric, Forge): Make block entities (beds, banners, signs, chests) visible from far away and not lag in rendering.

    • Somewhat recommanded mods:

  • Entity Culling (Fabric, Forge): Not render entities that you don't see.
  • Cull Less Leaves (Fabric, Forge*) (Has dependencies): Massively improve leaves performance (Mod is marked as 1.20.1 but works on 1.20.2 in practice).
  • Litematica (Fabric, Forge) (Has dependencies): Basically a must-have for making farms.
  • MiniHud (Fabric, Forge) (Has dependencies): It's not really "mini", but it's a super useful mod for display info bars and place virtual shapes in world.

    • How to install mods:

    1. Download Fabric loader here: https://fabricmc.net/use/installer
    2. Open the downloaded .jar file (fabric-installer-x.x.x.jar).
      • (if prompted about unidentified developer on Mac, right click and then select Open With > JavaLauncher.
      • If unable to open because you haven't installed Java, install JRE (Java RunTime Environment) here: https://www.oracle.com/java/technologies/downloads/ (Download the Installer not the Compressed Archive).
    3. In the Fabric Installer window, make sure to select 1.20.2 for Minecraft Version.
    4. Check Create profile.
    5. Click Install.
    6. You will be installing the mods by dragging them into the <minecraft>/mods/ folder, where <minecraft> is the Minecraft game folder, to find it:
      • On Windows: Press Win+R, type %appdata%\.minecraft, press Ok.
      • On macOS: Go to ~/Library/Application Support/minecraft/, you can use Cmd+Shift+G in finder to go to a directory, note that ~ just means your home directory (User/username).
      • On Linux: Go to ~/.minecraft/.
    7. In your Minecraft game folder, create a mods/ folder (Fabric will have created it for you if you have launched the game before), this is where you will drag all your mods into.
    8. Install Fabric API, this is a mod that virtually all mods depends on.
      • Download the .jar file from https://modrinth.com/mod/fabric-api, make sure to select the 1.20.2 version!
      • Move the .jar file into the aforementioned <minecraft>/mods/ folder.
    9. Install any other mods just as you installed the Fabric API mod.
