info.mc.shitfuck.pro

How to play on the mc.shitfuck.pro server:

  • Whitelist: Contact me on my socials or email kerbal255@outlook.com for whitelist.
  • Game version: Mineraft Java 1.20.2 (because some server-side performance mods aren't compatible with 1.20.4 yet)
  • No Chat Reports mod: The server has it, it's not required on your side, but it's highly recommanded.

    • Rules:

  • Freecam is allowed but only for working on your own buildings/farms, don't use it for PVP.
  • Placement mods are allowed but obviously to some degree (like bridging mod), so stuff like Litematica Printer is obviously not ok.
  • Don't mess with other's buildings.

    • Recommanded mods:

    (Make sure to use the 1.20.2 version!)

    (Forge* means the mod does not come with Forge support but there is an unofficial Forge port.)

  • Mod Menu: Adds mod menu to Fabric.
  • No Char Reports (Fabric, Forge): Disable chat reports.
  • Sodium (Fabric, Forge*): Massively enhancing performance.
  • Sodium Extra (Fabric, Forge*): Detailed render settings like fog distance.
  • Enhanced Block Entities (Fabric, Forge): Make block entities (beds, banners, signs, chests) visible from far away and not lag in rendering.

    • Somewhat recommanded mods:

  • Entity Culling (Fabric, Forge): Not render entities that you don't see.
  • Cull Less Leaves (Fabric, Forge*) (Has dependencies): Massively improve leaves performance (Mod is marked as 1.20.1 but works on 1.20.2 in practice).
  • Litematica (Fabric, Forge) (Has dependencies): Basically a must-have for making farms.
